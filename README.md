# Customer Churn Prediction (KNN)

A machine learning project that predicts customer churn using the K-Nearest Neighbors (KNN) algorithm.

## Features
- Data preprocessing using Pandas
- Train/Test split for model evaluation
- KNN model training with different values of K
- Accuracy comparison between training and testing sets
- Visualization of results using Matplotlib

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Learning Outcomes
- Implemented a classification model using KNN
- Practiced model evaluation and hyperparameter tuning
- Visualized model performance
